🏗 系统架构概览

Embedded Software Develop Agents Team（Driver HAL 路由可视化中心）是一套面向车规级嵌入式驱动开发的 AI Agent 团队管理平台，基于 Python Flask + 原生 JavaScript 构建，通过 WebSocket 实现实时通信。

📐 分层架构详解

前端层

纯原生 JavaScript（无框架），单页应用（SPA）架构，文件约 8500 行。

  • 资源池视图 — Agent/Skill/Rules/Knowledge/Tools 卡片网格
  • 项目管理视图 — 项目列表、CLI 绑定、团队成员管理
  • 路由可视化 — SVG 动态渲染指令→Agent→Skills 链路
  • 配置编辑器 — YAML 前置信息实时编辑
  • WebSocket 客户端 — 实时文件变更推送

后端层

Python Flask + Flask-SocketIO，单文件约 2800 行（gui/server.py）。

  • REST API — 项目/Agent/Skill CRUD 接口
  • ProjectManager — 项目配置持久化（projects_config.json）
  • CLI Detector — 自动识别本机安装的 Claude/Gemini/OpenAI 等工具
  • deploy_to_team — Agent 部署时自动复制关联资源
  • _copy_related_files — 解析 .md 前置 YAML + 章节代码块

🗂 开发资源池

开发资源池是全局共享的 Agent/Skill/Rule/Knowledge/Tool 仓库，位于项目根目录，与具体项目解耦。

Agents

10 个专家 Agent

每个 Agent 是一个 .md 文件，包含 YAML 前置信息（role、skills、tools）和 Markdown 章节（workflows、rules、knowledges）。

Pmic Agent Bridge Driver Safety Agent Mcal Agent

Skills

17 个技能包

每个 Skill 是一个目录，含 SKILL.md。对应 AUTOSAR 模块或硬件外设驱动开发规范。

TLF35584 SPI I2C Safetypack MCU

Rules & Knowledge

23 规则 33 知识项

编码规范（MISRA-C、AUTOSAR 命名）、ISO 26262 安全标准、芯片数据手册、开发流程文档。

项目是代码仓库的工作目录，系统支持新建或导入已有目录。

🔧 CLI 工具管理

CLI 工具是实际执行 AI 任务的命令行工具（Claude Code、Gemini CLI 等）。每个 CLI 在项目中对应一个隐藏目录（如 .claude/、.gemini/），Agent 和 Skill 文件需部署到该目录下才能被 CLI 识别。

💡 选择 CLI 后，左侧 Agents/Skills 列表将自动刷新为该 CLI 目录下的文件，而非资源池中的文件。

👥 组建项目团队

「组建项目团队」功能用于向当前选定 CLI 的工作目录批量添加 Agent 和 Skill，并自动解析依赖关系复制关联文件。

自动复制依赖文件逻辑

当添加一个 Agent 时，系统自动执行 _copy_related_files()：

1. 解析 agent.md 的 frontmatter YAML → 提取 skills/tools/rules/knowledges 键值
2. 扫描 agent.md 正文所有 ```yaml 代码块 → 提取引用路径
3. 在资源池中查找对应文件/目录
4. shutil.copy2/copytree 到项目 CLI 目录

🗺 智能路由

在主界面底部「路由关系」输入框中输入开发指令，点击「路由」按钮，系统自动匹配最合适的 Agent，并展示其关联的 Skills 和 Rules 链路图。

💡 路由结果也可通过 REST API 推送：POST /api/push_route {"instruction":"..."}，适合 CI/CD 集成和 Claude Code Hooks。

🔌 REST API 参考

项目管理 API

GET/api/projects — 获取所有项目列表
POST/api/projects — 新建项目 {name, path}
GET/api/project/<id> — 获取项目详情
POST/api/project/<id>/agent — 添加 Agent {src_path, folder}
POST/api/project/<id>/skill — 添加 Skill {src_path, folder}
DELETE/api/project/<id>/agent/<name> — 删除项目 Agent

资源池 API

GET/api/agents — 获取资源池所有 Agents
GET/api/skills — 获取资源池所有 Skills
GET/api/rules — 获取资源池所有 Rules
GET/api/knowledge — 获取资源池所有 Knowledge

路由 API

POST/api/push_route — 推送路由指令 {"instruction":"..."}
curl -X POST http://localhost:5000/api/push_route \
  -H "Content-Type: application/json" \
  -d '{"instruction":"TLF35584 PMIC 驱动初始化"}'

文件操作 API

GET/api/file?path=... — 读取文件内容
PUT/api/file — 保存文件内容 {path, content}
GET/api/cli/scan/<id> — 扫描项目 CLI 工具
